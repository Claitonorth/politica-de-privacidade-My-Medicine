Política de Privacidade — My Medicine / Meu Remédio

Aplicativo: My Medicine / Meu Remédio — Empresa: CHO App — Contato: suporte.choapp@gmail.com

1. Introdução

Esta Política de Privacidade descreve como a CHO App realiza o tratamento de dados pessoais e dados sensíveis dos usuários do aplicativo My Medicine / Meu Remédio, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD) e com as políticas da Google Play Store.

2. Público‑Alvo e Abrangência

O aplicativo é destinado ao público geral, incluindo adultos e idosos, podendo ser utilizado por usuários localizados no Brasil e em outros países.

3. Dados Pessoais Coletados

3.1 Dados fornecidos pelo usuário

  • Nome
  • Endereço de e‑mail
  • Data de nascimento
  • Sexo
  • Informações sobre medicamentos

3.2 Dados coletados automaticamente

  • Identificadores do dispositivo (ex.: Android ID)
  • Dados de uso do aplicativo

4. Dados Sensíveis

O aplicativo coleta dados sensíveis relacionados à saúde, tais como:

  • Nome de medicamentos
  • Dosagens
  • Condições médicas

Esses dados são sincronizados e armazenados em infraestrutura de nuvem fornecida pelo Firebase / Google Cloud.

5. Finalidade do Tratamento

Os dados são tratados para:

  • Autenticação do usuário
  • Funcionamento das funcionalidades principais
  • Sincronização e armazenamento das informações
  • Suporte ao usuário

6. Base Legal

O tratamento ocorre com base:

  • No consentimento do titular
  • Na execução do contrato de uso do aplicativo

7. Compartilhamento de Dados

Os dados podem ser compartilhados com:

  • Firebase Analytics (análise de uso)
  • Google AdMob (exibição de anúncios)

⚠️ Dados sensíveis de saúde não são utilizados para personalização de anúncios, segmentação comportamental ou perfilhamento publicitário.

8. Armazenamento e Retenção

Os dados são armazenados em servidores do Firebase / Google Cloud e permanecem armazenados mesmo após a desinstalação do aplicativo, sem prazo de retenção previamente definido, até que o usuário solicite sua exclusão.

9. Direitos do Titular

O usuário pode solicitar:

  • Acesso
  • Correção
  • Exclusão
  • Revogação do consentimento

Solicitações via: suporte.choapp@gmail.com

10. Segurança da Informação

Os dados são transmitidos por meio de conexões seguras (HTTPS/TLS) fornecidas pela infraestrutura do Firebase. O acesso é protegido por mecanismos de autenticação.
